VR Fitness Gym Log

Reflex team: Jody, Cam, Maddie, Kevin, Allen

Objective

We are creating a handheld fitness device that allows users to set target reps and log completed reps, velocity, etc.. The data will be sent to a site that users can log into to manage preferences and view their stats. This project is being worked on in conjunction with another course project from the concurrent Hacking the Holodeck IAP.

Our immediate tasks are:

  • Get acceleromater to velocity, rotational velocity, and count reps
  • Collect data from bluetooth sensor and output to webpage
  • Build page to manage target reps & goals

Future tasks:

  • If time permits, use input from velocity & reps to output visual or audio confirmation
  • Collect force
